# ROIS
Recruitment and Onboarding Information System
Developed by C5

## Setting Environment
1. git clone 
2. composer install
3. npm install

## Run The Code
1. buka cmd, npm run start 
2. buka cmd lain, php -S localhost:8000 -t public

## Database
1. Buat database baru bernama c5_rois di phpMyAdmin
2. import sql dump

## Prerequisite
1. composer artisan
2. npm
3. xampp
